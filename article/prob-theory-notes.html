基本概念

样本空间 (Sample Space)

一个随机试验的所有可能结果的集合，通常用Ω表示。

事件 (Event)

样本空间的子集，即某些结果的集合。

示例：掷骰子

样本空间: Ω = {1, 2, 3, 4, 5, 6}

事件A（掷出偶数）: A = {2, 4, 6}

概率公理

柯尔莫哥洛夫公理

  1. 非负性：对于任何事件A，P(A) ≥ 0
  2. 规范性：P(Ω) = 1
  3. 可列可加性：对于互斥事件A₁, A₂, ...，有P(∪Aᵢ) = ΣP(Aᵢ)

概率性质

  • P(∅) = 0
  • P(Aᶜ) = 1 - P(A)
  • 如果A ⊆ B，则P(A) ≤ P(B)
  • P(A ∪ B) = P(A) + P(B) - P(A ∩ B)

条件概率

P(A|B) = P(A ∩ B) / P(B)，其中P(B) > 0

贝叶斯定理

P(A|B) = P(B|A) × P(A) / P(B)

示例：疾病检测

假设一种疾病的患病率为1%，检测准确率为99%。如果一个人检测结果为阳性，求他实际患病的概率。

解：使用贝叶斯定理，P(患病|阳性) ≈ 50%

随机变量

随机变量

定义在样本空间上的实值函数，将每个结果映射到一个实数。

随机变量类型

  • 离散随机变量：取值可数的随机变量
  • 连续随机变量：取值充满一个区间的随机变量

示例

离散：掷骰子的点数

连续：某地区年降雨量

概率分布

常见离散分布

  • 伯努利分布：单次试验，成功概率p
  • 二项分布：n次独立伯努利试验的成功次数
  • 泊松分布：单位时间内随机事件发生的次数

常见连续分布

  • 均匀分布：在区间[a,b]上等可能取值
  • 正态分布：钟形曲线，广泛应用
  • 指数分布：描述独立随机事件发生的时间间隔
正态分布概率密度函数：f(x) = (1/√(2πσ²)) × e^(-(x-μ)²/(2σ²))

数学期望

期望值

随机变量取值的加权平均，权重为概率。

离散随机变量：E[X] = Σ xᵢ × P(X=xᵢ)
连续随机变量：E[X] = ∫ x × f(x) dx
方差：Var(X) = E[(X - E[X])²] = E[X²] - (E[X])²

示例：掷骰子的期望

E[X] = (1+2+3+4+5+6)/6 = 3.5

大数定律

大数定律

当试验次数足够多时，样本均值收敛于总体均值。

limn→∞ P(|X̄n - μ| < ε) = 1

示例：硬币投掷

随着投掷次数增加，正面朝上的频率越来越接近0.5
